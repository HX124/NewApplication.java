常见问题解答

1. 进入"项目管理"页面，点击右上角的"新建项目"按钮

2. 填写项目基本信息：项目名称、产品/过程名称、负责人等

3. 选择项目模板（可选）

4. 配置评分规则和风险阈值

5. 点击"创建"按钮完成项目创建

系统支持Excel格式的数据导入和导出：

导入：在项目详情页面，点击"导入数据"按钮，选择Excel文件，系统会自动解析数据

导出：在项目详情页面，点击"导出"按钮，选择导出格式（Excel/PDF）

1. 进入项目设置页面

2. 点击"团队管理"标签

3. 添加团队成员，分配相应权限：

  • 查看者：只能查看项目内容
  • 编辑者：可以编辑项目内容
  • 审核者：可以审核和批准变更
  • 管理员：完全控制权限
